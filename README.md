## Getting Started

This project require NodeJS v12.

### Install
Clone this project

```bash
> git clone https://github.com/XyroMawwari/Xyro-Bot.git
> cd Xyro-Bot
```

Install the dependencies:

```bash
> npm install 
> npm install gify-cli -g
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```
